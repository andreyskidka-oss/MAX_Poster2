# MAX Poster Bot v3.4 — Документация

## Обзор

MAX Poster — бот для мессенджера MAX, публикующий посты в каналы с поддержкой кнопок, форматирования, планирования, закрепления и пакетной отправки.

## Функции

### Регистрация
- /start → токен (32 символа, кликабельный)
- Автоопределение каналов (событие bot_added)

### Каналы
- Автообнаружение, проверка прав каждые 5 мин
- Карточка канала: статус, помощники, кнопки

### Кнопки (3 набора на канал)
- 🔖 Анонсы бота — для постов из @best_anons_bot
- 🆓 Свои анонсы — для постов с ценой + картинкой
- ✍️ Ручной постинг — для «Создать пост»
- Библиотека, мультивыбор, порядок, расположение 1/2/3 в ряд, превью

### Создание постов
- Пакетный режим, мультивыбор каналов
- 📤 Отправить сейчас / ⏰ Запланировать пост / 📌 Запланировать закреп

### Планирование (v3.4)
- Дата: ДД.ММ или ДД.ММ.ГГГГ — валидация, контроль прошлых дат
- Время: ЧЧ:ММ — валидация, контроль прошлого времени
- Превью расписания, подтверждение
- Закреп на неделю: 7 постов → по дням
- Scheduler: проверка каждые 30 сек

### Закрепление (v3.4)
- PIN API: PUT /chats/{chatId}/pin
- Автозакрепление, обновление URL в привязанных кнопках
- Логирование: Pin OK / PIN FAILED

### Очередь (v3.4)
- 📋 Очередь отправки в главном меню
- Тип, дата/время, каналы, превью, отмена

### Форматирование
- HTML: жирный, курсив, подчёркнутый, зачёркнутый, ссылки
- Цитаты → blockquote
- Номер поста (6 цифр) → машинописный + подчёркнутый

### Тарифы
- API формат: {"status": "OK", "accs": [...]}
- Безопасный режим, 3 текста напоминаний

### Надёжность
- Exponential backoff (1→2→4→8→16→30 сек)
- Fallback без кнопок при 400
- Per-channel error handling
- Docker restart + healthcheck + autoheal + watchdog

## REST API (14 эндпоинтов)

Авторизация: `Authorization: Bearer <token>`

| Метод | Путь | Описание |
|-------|------|----------|
| GET | /health | Статус |
| GET | /channels | Каналы пользователя |
| POST | /post | Отправка поста |
| POST | /edit | Редактирование (24ч) |
| POST | /delete | Удаление поста |
| POST | /delete-announcement | Удаление анонса |
| GET | /buttons | Глобальные кнопки |
| POST | /buttons | Задать глобальные |
| GET | /buttons/{id} | Кнопки канала |
| POST | /buttons/{id} | Задать кнопки канала |
| DELETE | /buttons/{id} | Удалить кнопки канала |
| GET | /buttons/{id}/sets | Все наборы |
| GET | /buttons/{id}/{type} | Набор по типу |
| POST | /buttons/{id}/{type} | Задать набор |

## Ограничения MAX API
- Rate limit: 30 rps
- Цитаты теряются при пересылке
- HTML format для форматирования
- Теги: b, i, u, s, code, a, blockquote, em, pre
